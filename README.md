# サーバー崩壊ラプソディ 🖥️💥

**Server Collapse Rhapsody** - ITエンジニアのためのクソゲー

タップしてサーバーに負荷を与え、崩壊させよう。毎回違う体験が待っている。

## 遊び方

1. 画面をタップ/クリックしてサーバーに負荷を与える
2. メーターが上がっていく
3. ある瞬間、サーバーが崩壊する
4. 崩壊スコアを確認
5. 再起動して繰り返す

## 特徴

- **完全ランダム**: 同じパターンは二度と出ない
- **レトロスタイル**: 8色パレット、ドット絵風
- **6種類の崩壊演出**: 爆発、フリーズ、散乱、BSOD、消滅、メルトダウン
- **8bitサウンド**: Web Audio APIで動的生成
- **ITエンジニアあるあるログ**: `Segmentation fault`、`It works on my machine`など

## ランダム要素

- CPU負荷: 10〜300%
- メモリ使用量: 128MB〜32GB
- ネットワーク遅延: 0ms〜5000ms
- 崩壊トリガー条件: 6種類の条件からランダム選択
- 崩壊演出: 6パターン
- テキストログ: 30種類以上
- サウンド: 4パターン（無音含む）

## 技術スタック

- HTML / CSS / JavaScript (Vanilla)
- Canvas API
- Web Audio API
- フレームワークなし

## デプロイ

GitHub Pagesで直接デプロイ可能。`index.html`をルートに配置するだけ。
